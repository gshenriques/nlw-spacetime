Esse é um projecto web responsivo de uma cápsula do tempo para ixibir memórias em uma tinha do tempo.

Esse projecto foi desenvolvido durante o NLW - SPACETIME com as seguintes tecnologias:

- HTML
- CSS
- Git e Github
